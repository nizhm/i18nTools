## 使用说明：

### 1. 初始化项目环境及安装依赖
```
npm run init
```

### 2. 所有应用在apps文件夹
<br/><br/><br/>
## 主要应用： 

### 1.excelKeyFunCreator

  生成excel函数，此函数自动读取英文翻译生成key；
  
### 2.excelToJs

  读取excel某一表格的数据，并生成js文件；
  
### 3.autoReplacer

  自动化i18n替换vue文件
  
### 4.chineseInspector
  检索指定文件中的中文字符串
